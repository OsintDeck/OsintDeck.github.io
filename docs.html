Documentación de OSINT Deck

Guía completa para implementar y configurar el plugin

v1.0 Ver en GitHub

Introducción

OSINT Deck es un plugin de WordPress diseñado para centralizar y organizar herramientas OSINT en un mazo dinámico. El sistema detecta automáticamente el tipo de dato ingresado y activa las herramientas correspondientes, permitiendo análisis rápidos, ordenados y reproducibles en un entorno unificado.

Nota: Esta documentación describe el estado actual del plugin (versión 1.0) en la rama principal del repositorio. Cuando se publiquen nuevas versiones estables, esta página se actualizará con los cambios correspondientes.

Sobre el Proyecto

OSINT Deck nace como una iniciativa del equipo de OSINT.com.ar, un blog especializado en inteligencia de fuentes abiertas y ciberseguridad. La herramienta fue desarrollada para optimizar los flujos de trabajo de investigación que utilizamos diariamente en nuestras publicaciones y análisis.

Mantente actualizado: Visita osint.com.ar para leer artículos sobre técnicas OSINT, herramientas, casos de estudio y las últimas novedades en ciberseguridad.

Características Principales

Detección automática del tipo de dato

El plugin identifica automáticamente el contenido ingresado, soportando:

  • Email, Dominio / subdominio, URL, IP v4 / v6
  • ASN, MAC, UUID, Hashes (MD5, SHA1, SHA256)
  • Teléfonos, Coordenadas, Wallets BTC / ETH
  • Claves PGP, Archivos conocidos, Username / alias
  • Nombre completo, ZIP / Código postal, Paquetes com.*.*

Mazo dinámico de herramientas

  • Cartas individuales por herramienta
  • Botón principal "Analizar {input}"
  • Acciones extra: copiar, abrir, compartir
  • Filtros por categoría, licencia y tipo de acceso
  • Badges automáticos: Nueva, Popular, Reportada, Recomendada

Métricas internas

Las métricas se generan diariamente y se almacenan en osd_tool_metrics. Incluyen clicks_7d, reports_7d, created_at, last_input_type.

Shortcode

Implementa el mazo en cualquier página o entrada:

[osint_deck]
[osint_deck category="dominios" access="gratuito" limit="20"]

Parámetros disponibles:

  • category: filtra por categoría de herramienta (ej. "dominios", "red").
  • access: tipo de acceso (ej. "gratuito", "registro", "pago").
  • limit: cantidad inicial de resultados (por defecto 20).
Tip: Puedes combinar múltiples parámetros para crear vistas personalizadas del mazo según tus necesidades.

Instalación

Instalación manual

  1. Descargar el repositorio desde GitHub.
  2. Copiar la carpeta del plugin dentro de: wp-content/plugins/osint-deck
  3. En el panel de WordPress ir a: Plugins → OSINT Deck → Activar.

(Opcional) Importar herramientas usando los JSON de template-de-herramientas/*.json.

Vía WP-CLI

wp plugin activate osint-deck
Importante: Asegúrate de tener WordPress 6.4+ y PHP 8.0+ antes de instalar el plugin.

Validación de dominios y TLDs

  • Lista IANA local en assets/data/tlds-alpha-by-domain.txt.
  • Tarea semanal: osd_refresh_tlds_weekly.
  • Validación offline mediante osd_is_valid_domain() (sin llamadas externas para proteger la privacidad).

Rate limiting

Implementado en osd_user_event para prevenir abusos:

  • 60 acciones por minuto por IP.
  • 1 reporte por herramienta por día.
  • Respuestas JSON consistentes (códigos como rate_limited, report_limit).

Administración

Funcionalidades del panel de administración:

  • Configuración de límites y opciones de seguridad.
  • CRUD (Crear, Leer, Actualizar, Borrar) de herramientas.
  • Importación y exportación en formato JSON.
  • Logs de usuario y logs administrativos.
  • Métricas resumidas por herramienta.

Hooks y tareas programadas

Cron jobs principales

  • osd_metrics_daily: Generación de estadísticas diarias.
  • osd_refresh_tlds_weekly: Actualización de la lista de TLDs.

AJAX

Públicos

  • osd_user_event: Registro de interacciones.
  • osd_check_tld: Validación asíncrona.

Administrativos

  • osd_metrics_summary
  • osd_tools_*
  • osd_logs_*
